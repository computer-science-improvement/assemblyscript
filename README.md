## AssemblyScript

---

### How to run this template:

```html
1. git clone ...
2. cd ./assemblyscript
3. npm install
4. npm asbuild
5. npm start
6. goto localhost:3000
```

---

### Installation:

```html
1. npm init
2. npm install --save-dev assemblyscript
3. npx asinit .
4. npm run asbuild
5. npm start
```
